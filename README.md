# Retail Audit Data Application

## Business Context
Retail audit data plays a critical role in identifying pricing discrepancies, shrink trends, and operational compliance gaps.

## Objective
Develop a lightweight analytical application to process audit data and surface actionable performance insights.

## Tools Used
- Python
- SQL
- Data validation techniques

## Key Outcomes
- Structured audit dataset for analysis
- Identified margin-impacting discrepancies
- Built reusable data processing workflow
